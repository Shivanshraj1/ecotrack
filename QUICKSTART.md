# 🚀 EcoTrack - Quick Start Guide

## Get Started in 60 Seconds

### Option 1: Direct Browser (Easiest)
Simply open `index.html` in any modern browser:
- Double-click the file
- Or drag it into your browser window

### Option 2: Local Development Server
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (with http-server)
npx http-server -p 8000

# PHP
php -S localhost:8000
```

Then visit: **http://localhost:8000**

## 🐛 Troubleshooting

**Issue**: Charts not rendering
- **Solution**: Make sure internet connection is active (Chart.js loads from CDN)

**Issue**: Map not loading
- **Solution**: Check internet connection (Leaflet.js + OpenStreetMap require network)

**Issue**: Can't log in
- **Solution**: Use demo credentials or create new account via Sign Up

**Issue**: Points not saving
- **Solution**: Don't use Private/Incognito mode (localStorage required)
