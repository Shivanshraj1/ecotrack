## 🚀 How to Use

### **Quick Start (2 Minutes)**
```bash
# Option 1: Direct browser
open index.html

# Option 2: Local server
python -m http.server 8000
# Visit http://localhost:8000
```

### **Demo Credentials**
```
Household: demo@eco.com / demo123
Business:  business@eco.com / business123
```

### **Test Flow**
1. Login with demo account
2. Navigate to "AI Scanner"
3. Upload any waste image
4. Click "Classify Waste"
5. Earn eco-points
6. View dashboard updates
7. Redeem rewards (if enough points)
8. Explore recycler map
9. Check pricing plans
